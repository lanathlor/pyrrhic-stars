# Mythic+ System

Keystone system for repeatable dungeon content:

- Keystone items with level and affix rolls
- Difficulty scaling: creature HP/damage multiplied by key level
- Timer: completion within time limit for keystone upgrade
- Affixes: periodic modifiers (e.g. Bolstering: when a creature dies, nearby creatures gain HP; Volcanic: random eruptions under ranged players; Flux Storm: increased Flux instability chance)
- Leaderboard: per-dungeon, per-key-level, tracked in PostgreSQL, displayed on companion web app
