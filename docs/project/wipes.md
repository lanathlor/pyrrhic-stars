# Alpha/Beta Wipes

Standard practice in MMO development. Players accept it when communicated well.

## What Gets Wiped

Characters, gear, currency, inventory (except cosmetics), crafting progression, quest progress, kill credits, active Overflux records.

## What Persists

- Account creation date and participation history
- Cosmetic rewards per season
- "Alpha Tester" / "Beta Season N" badges
- Patreon supporter cosmetics
- Glamour catalog unlocks
- Bug report and contribution credits

## Cadence

Align with planned season cadence (3-4 months). More frequent feels punitive; less frequent loses testing benefit.

## Use Cases

- Test bold balance changes
- Iterate on Overflux curve magnitude
- A/B test decay curves across consecutive seasons
- Verify seasonal transition mechanism robustly before launch

## Communication

- Sign-up agreement explicit about wipes
- Each wipe announced as event weeks ahead, reasons given
- Snapshot moments preserved (final-state as glamour)
