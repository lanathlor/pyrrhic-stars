# Marketing Strategy

## Core Message

"A co-op action game where every class plays a different genre."

Never say "I'm building an MMO" publicly until substantial content exists. Use "co-op action game" or "multiplayer dungeon crawler."

## Build in Public

Weekly 30-60 second clips showing one concrete thing working. Every week, no exceptions.

- Week 1: "FPS controller"
- Week 2: "Melee controller in the same game"
- Week 3: "Both fighting the same enemy"
- Week 4: "Added the Blade Dancer"
- And so on...

## Target Communities

Early: r/godot, r/indiegaming, r/gamedev (supportive of solo devs)
Later (with content): r/MMORPG, gaming media, content creators

## The Key Clip

5 players in a dungeon. Different gameplay modes visible simultaneously. One continuous 2-minute clip. Real gameplay, not scripted. Friends on voice chat reacting naturally. This clip proves everything at once.

## Steam Wishlists

Get the Steam page up during Phase 1. Wishlists convert to sales at 10-20%. 10,000 wishlists before launch = 1,000-2,000 first-week sales. Steam's algorithm promotes high wishlist velocity.
