# Monetization Plan

## Timeline

- Month 2: post clips, build Discord community
- Month 4: Patreon (3 EUR/month) for early access builds
- Month 6: itch.io early access (10-15 EUR)
- Month 8-12: Steam Early Access (15-20 EUR)
- Month 9+: cosmetic shop (skins, weapon models, never pay-to-win)

## Don't Build Early

- Custom launcher (use itch.io, then Steam)
- Payment system (use Patreon, then itch.io, then Steam)
- Website beyond landing page with Discord link
- Account system before Phase 2 (use Discord OAuth)
- Anti-cheat (PvE co-op with friends doesn't need it yet)

## Revenue Expectations (honest)

| Tier | Probability | Revenue (2-3 years) | Monthly peak |

| "It works but nobody came" | ~60% | 5,000-25,000 EUR | Negligible |

| Phenomenon | ~1-2% | Millions | Job-replacing |
